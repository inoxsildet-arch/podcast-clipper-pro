# Contribuindo

Contribuições são bem-vindas! Este é um projeto pessoal/portfólio, mas sugestões,
correções e ideias são sempre bem-vindas via Issues ou Pull Requests.

## Como contribuir

1. Faça um fork do repositório
2. Crie uma branch para sua alteração: `git checkout -b minha-feature`
3. Faça commit das suas mudanças: `git commit -m "Descrição clara da mudança"`
4. Envie para o seu fork: `git push origin minha-feature`
5. Abra um Pull Request descrevendo o que foi alterado e por quê

## Reportando problemas

Ao abrir uma Issue, inclua:
- Sistema operacional e versão do Python
- Passos para reproduzir o problema
- Mensagem de erro completa (se houver)
- Comportamento esperado vs. comportamento observado

## Estilo de código

- Siga o padrão já usado no projeto (nomes de função em português para lógica de
  domínio, docstrings explicando o propósito de cada método)
- Prefira clareza a brevidade
